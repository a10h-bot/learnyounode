babyAnimals is a curried function; it is designed for the first argument
to be ‘prefilled’ before the function itself is fully executed.
With this pattern, ‘koala’ can be bound to babyAnimals, and my love for animals other 
than elephants can easily be expressed.
Currying can be integrated with callbacks to create higher-order ‘factory’ functions.
